# BUG_REGISTRY.md — RorkParity

Convention matches Glitchfront's registry. One entry per confirmed, recurring, or
user-facing bug. WHETSTONE's routine catch-and-fix loops don't get an entry — only
patterns that recur across 3+ generations, or bugs that reached the user.

Format:

```
## RP-001
**Status:** open | fixed | wontfix
**Found:** <date>
**Agent/system:** which agent or which game system was involved
**Description:** what went wrong
**Root cause:** (once known)
**Fix:** (once fixed)
```
